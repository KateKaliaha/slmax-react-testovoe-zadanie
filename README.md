## Начало работы

```
git clone https://github.com/KateKaliaha/slmax-react-testovoe-zadanie.git
```

```
npm i
```

**Необходимо установить json сервер**

```
npm i -g json-server
```

**Запуск сервера с базой данных**

```
npm start server
```

**Запуск в dev режиме**

```
npm run dev
```

**Создание build и его запуск**

```
npm run build
npm run start
```
